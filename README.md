# Global Air Pollution Analysis Dashboard

## Overview
This project presents an interactive Power BI dashboard for analyzing global air pollution data. The dashboard provides insights into Air Quality Index (AQI) levels, pollutant distribution, and air quality trends across different countries and cities.

## Dataset
* Source: Kaggle
* Dataset: Global Air Pollution Dataset

## Tools Used
* Power BI

## Dashboard Features
* AQI (Air Quality Index) Analysis
* PM2.5, NO₂, CO, and Ozone Insights
* Country-wise and City-wise Comparisons
* Interactive Filters and KPI Cards
* Treemaps, Pie Charts, and Donut Charts

## Key Insights
* Identified regions with high AQI levels.
* Compared pollutant concentrations across locations.
* Visualized air quality categories and pollution trends.
